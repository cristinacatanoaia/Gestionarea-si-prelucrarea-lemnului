# Gestionarea-si-prelucrarea-lemnului
Aplicatia are la baza prelucrarea lemnului. De acolo vor fi urmatoarele: adaugarea unei liste de clienti, cantitatea de lemn neprelucrat din care vor rezulta mai multe tipuri de debitare si procesare(cherestea, lambriu, etc.) .Cantit. de lemn prelucrat va fi vanduta si se va tine evidenta cantitatii si tipului de lemn vandute fiecarei firme.
